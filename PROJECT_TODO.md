## ✅ Milestone 2：核心功能补全（Feature Completeness）

**目标**：补齐在线听歌核心闭环（播放、搜索、登录态）。

### 模块：播放增强（Playback）
- [x] **Task 2.1.1：播放历史记录（Play History）**
  - 记录最近 100 首播放歌曲并持久化。
  - 首页新增“播放历史”页签，支持快速回播与清空历史。
- [x] **Task 2.1.2：音质切换选择（Quality Selector）**
  - 播放栏增加标准/极高/无损音质切换。
  - 播放链路按偏好音质请求 URL，并进行多级回退。
- [x] **Task 2.1.3：歌词平滑滚动与偏移校准**
  - 歌词支持 ±0.5s 偏移、重置与持久化。
  - 高亮行改为容器平滑居中滚动，提升观感。

### 模块：搜索与登录（Discovery & Auth）
- [x] **Task 2.2.1：搜索建议与历史**
  - 新增本地搜索历史存储。
  - 搜索下拉支持建议词、历史词回填、历史清空。
- [x] **Task 2.2.2：搜索结果跨平台去重**
  - 基于“歌曲名 + 歌手名”跨平台去重。
  - 引入音质优先策略，优先保留质量更高版本。
- [x] **Task 2.2.3：登录自动续期与过期提醒**
  - 周期校验网易云/QQ 登录有效性。
  - 网易云尝试后台续期；失败/过期时在首页展示提醒。

> 完成记录：2026-02-24（M2 2.1.1 ~ 2.2.3 完成）
